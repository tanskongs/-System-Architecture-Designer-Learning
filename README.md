# 备考计划

## 核心结论：

采用“先主后辅、阶段聚焦”策略，2025年11月-2026年5月主攻系统架构师，雅思同步打基础；2026年6月-12月全力冲刺雅思，架构师知识按需复盘，适配事业单位在职节奏，兼顾效率与可持续性。

### 第一阶段：架构师主攻+雅思基础（2025.11-2026.5，共7个月） 

核心目标：拿下系统架构师三科（综合知识、案例分析、论文），雅思筑牢词汇+语法根基，避免双线发力导致精力分散。

 #### 架构师学习路线（每日2-2.5小时，周末4-5小时）

1. 基础夯实期（2025.11-2026.1，3个月）

   - 以《系统架构设计师教程（第4版）》为核心，搭配《软件工程》《计算机组成原理》补充基础。
   - 用思维导图拆解高频考点，比如微服务架构的拆分原则、通信方式等，构建知识框架。 - 每天刷30-50道综合知识真题，重点突破架构设计理论、计算机基础知识模块。 

2.  强化突破期（2026.2-2026.3，2个月）

   - 综合知识聚焦新技术应用（云计算、AI、信创适配）和系统安全模块，总结易混考点对比表。 
   - 案例分析专攻架构优化、技术选型题型，提炼“问题定位→技术依据→方案设计→风险评估”答题模板。
   - 论文锁定2个擅长主题（如微服务、高并发架构），每周积累1个项目案例素材，记录量化数据（如响应延迟优化、并发量提升）。

3. 刺模拟期（2026.4-2026.5，2个月）

   - 按考试时间全真演练，每天1套真题（2.5小时综合知识+1.5小时案例分析+2小时论文）。

   - 复盘错题标注知识点归属，补充2025-2026年新技术热点（如AI驱动架构、边缘计算）。 
   - 论文每周完成1篇完整写作，优化结构完整性与技术深度，避免模板化表述。

 #### 雅思基础同步推进（每日1-1.5小时，碎片化时间利用）

- 词汇：用“主题分类法”记忆核心词汇，覆盖教育、科技、环境等8大场景，每天10-15个，结合APP复习巩固。

- 语法：重点攻克长难句解析，每周梳理1个语法难点（如定语从句、倒装句），结合简单阅读材料应用。

- 听力：每天15-20分钟泛听（BBC新闻、雅思真题音频），培养语感，无需刻意追求正确率。

 ### 第二阶段：雅思主攻+架构师复盘（2026.6-2026.12，共7个月）

 核心目标：雅思四科专项突破+全真模拟，架构师知识通过碎片化复盘保持记忆，应对后续职业应用。

#### 雅思学习路线（每日2.5-3小时，周末5-6小时）

1. 基础强化期（2026.6-2026.7，2个月）

   - 词汇升级：背诵雅思高频同义替换词，每天20个，结合阅读真题语境记忆。

   - 专项训练：上午1小时听力精听（用《雅思王听力真题语料库》），下午1小时阅读定位技巧练习。

   - 写作/口语：每周2篇小作文练格式（图表题），每天10分钟口语话题跟读（日常场景如工作、爱好）。

2. 专项突破期（2026.8-2026.10，3个月）

听力：重点训练同义替换识别，错题按题型分类（填空、选择、匹配），针对性补强。 

阅读：掌握快速定位、逻辑关系分析技巧，每天完成1篇真题，总结错题原因（如逻辑误判、细节遗漏）。

写作：建立大作文框架（引言+论点+论据+结论），每周2篇完整写作，优化表达流畅度。 

口语：参考当季题库，每天1个话题演练，注重逻辑连贯和自然表达，录音回听修正发音。 

冲刺模拟期（2026.11-2026.12，2个月） 

- 用《剑桥雅思真题16-18》全真模拟，每天1套完整考试（听力+阅读+写作+口语），严格把控时间。
- 针对薄弱项集中突破，比如写作逻辑不清晰就强化论点组织，口语发音不准就加强跟读模仿。 
- 调整考试状态，熟悉流程，每周总结1次进步与不足，优化时间分配策略。

#### 架构师知识复盘（每周1-2小时）

- 月梳理1个核心模块（如架构评估方、系统性能优化），避免知识遗忘。
- 关注行业动态，将架构师理论与实际工作结合，提升应用能力，同时为职业发展储备能量。

### 在职备考关键保障

- 时间管理：工作日利用早间1小时+晚间2小时集中学习，通勤时间用APP背单词、听听力；周末预留完整时间段进行真题演练和深度复习。
- 精力分配：35岁需注重劳逸结合，每周留1天休息时间，避免过度疲劳；学习时关闭无关软件，提高专注度。 
- 资料选择：架构师锁定官方教材+近5年真题+行业热点报告；雅思选用《剑桥雅思真题集》+核心词汇书+听力语料库，拒绝冗余资料。 要不要我帮你整理一份**分阶段详细学习打卡表**，包含每日具体任务、时间分配和复盘模板，让备考更有条理？
